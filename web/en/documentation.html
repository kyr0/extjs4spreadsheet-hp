Documentation

To deliver a great product and support the developers using Ext JS 4 SpreadSheet I focused on writing a good documentation too.

The documentation is available as comments the source code and as a HTML5 application generated by the great JavaScript documentation tool JSDuck.

A word on software quality

As an experienced, long-term Ext JS developer & author I really do know the anti-patterns. Complex, non-customizable and bad documented components. Components that aren't supported by the original component author and make use of deprecated APIs.

Ext JS 4 SpreadSheet is fully customizable, event-driven (hookable), well documented, uses state-of-the-art Ext JS 4 APIs. I'm also offering commercial support.

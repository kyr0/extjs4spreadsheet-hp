Ext JS 4 - SpreadSheet

Ext JS 4 SpreadSheet is a highly customizable Microsoft Excel®-like component for Ext JS 4.

It is build on the standard Ext JS 4 grid component and supports Excel®-like:

  • – a calc selection model (mouse and keyboard)
  • – copying & pasting from/into Microsoft Excel®
  • – cell editing and custom rendering
  • – granular control over editing and selecting
  • – and many more!

It works in any browser Ext JS 4 supports. It integrates with any plugin and grid feature bundled with Ext JS 4. It's implementation is fully evented and therefore customizable!

While development I focused on excellent code quality and documentation too.

Live Demo – Source Code

Integrating the Ext JS 4 SpreadSheet is easy: 



new Spread.grid.Panel({
     title: 'Users (generated sample data)',
     store: localDataStore,
     editable: true,
     columns: [{
         xtype: 'spreadheadercolumn',
         header: 'ID'
     }, {
         header: 'Last name<br>(editable)',
         renderer: function (value) {
             return '<i>' + value + '</i>';
         },
         dataIndex: 'lastname',
         flex: 1
     }, {
         allowedEditKeys: '0123456789.',
         header: 'Age<br>(editable)',
         dataIndex: 'age',
         xtype: 'numbercolumn'
     }, {
         header: 'Birthday <br>(not editable)',
         dataIndex: 'birthday',
         xtype: 'datecolumn',
         editable: false
     }]
 })
